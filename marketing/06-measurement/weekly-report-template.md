## 📖 HƯỚNG DẪN SỬ DỤNG

### Quy trình cập nhật hàng tuần

```
Thứ 4-5:  Team members cập nhật số liệu từ các platform
Thứ 5:    Ads + Content gửi số liệu cho Marketing Manager
Thứ 6 AM: Marketing Manager tổng hợp vào báo cáo
Thứ 6 14:00: Weekly Review Meeting (30 phút)
           → Review báo cáo → Discuss issues → Set action items tuần sau
Thứ 6 PM: Finalize báo cáo + gửi cho stakeholders
```

### Data Sources

| Metric | Lấy từ đâu |

| Traffic, Sessions, Bounce Rate | GA4 → Reports → Acquisition |
| Organic keywords, impressions | Google Search Console → Performance |
| Orders, Revenue, AOV | WooCommerce → Reports |
| Ads performance | Meta Ads Manager / Google Ads / TikTok Ads |
| Social metrics | Meta Business Suite / TikTok Analytics / Zalo OA |
| Email metrics | Email platform (Mailchimp / SendGrid) |

| Màu | Ý nghĩa | Khi nào dùng |
