## 2. Trust Signals Cần Thiết

### 2.1. Chứng nhận & Thông tin doanh nghiệp
- [ ] Hiển thị giấy chứng nhận đại lý ủy quyền Xiaomi (nếu có)
- [ ] Mã số thuế doanh nghiệp ở footer
- [ ] Địa chỉ văn phòng/kho hàng cụ thể
- [ ] Số điện thoại hotline (hiển thị to ở header)
- [ ] Email liên hệ chính thức (domain @xiaomi247.com)

### 2.2. Chính sách rõ ràng
- [ ] Trang bảo hành chi tiết với quy trình cụ thể
- [ ] Trang đổi trả với điều kiện rõ ràng
- [ ] Trang vận chuyển với bảng phí & thời gian cụ thể
- [ ] Chính sách bảo mật thông tin
- [ ] Điều khoản sử dụng

### 2.3. Social Proof
- [ ] Review khách hàng thực trên từng sản phẩm
- [ ] Testimonial video từ khách hàng
- [ ] Số đơn hàng đã bán (counter)
- [ ] Rating trung bình sản phẩm
- [ ] Badge "Đã xác minh mua hàng" cho reviewer

### 2.4. Video & Visual Trust
- [ ] Video unboxing sản phẩm mới
- [ ] Video so sánh hàng chính hãng vs hàng nhái
- [ ] Hình ảnh kho hàng thực tế
- [ ] Hình ảnh đóng gói cẩn thận
- [ ] Livestream bán hàng trên Facebook/TikTok

## 3. Trust Badges Đề Xuất

### 3.1. Badges chính (hiển thị trên header/product page)

```
🛡️ Chính hãng 100%          — Cam kết hàng chính hãng Xiaomi
🔧 Bảo hành 12-24 tháng     — Bảo hành chính hãng toàn quốc
🚚 Giao hàng toàn quốc      — Ship COD, kiểm tra trước khi nhận
🔄 Đổi trả 7 ngày           — Đổi trả miễn phí nếu lỗi nhà sản xuất
🔒 Thanh toán an toàn        — Hỗ trợ nhiều hình thức thanh toán
```

### 3.2. Vị trí hiển thị badges
| Vị trí | Badges hiển thị | Ưu tiên |
|---|---|---|
| Header (sticky) | Hotline + "Chính hãng 100%" | ⭐⭐⭐ |
| Dưới giá sản phẩm | Tất cả 5 badges dạng icon nhỏ | ⭐⭐⭐ |
| Sidebar product page | 5 badges dạng list với mô tả | ⭐⭐ |
| Footer | Chứng nhận + phương thức thanh toán | ⭐⭐ |
| Trang chủ (hero section) | 3 badges chính dạng banner | ⭐⭐⭐ |
| Checkout page | "Thanh toán an toàn" + "Đổi trả 7 ngày" | ⭐⭐⭐ |

### 3.3. Thiết kế badges
- **Style:** Flat design, icon + text ngắn
- **Màu:** Xanh dương (#1E88E5) cho trust, cam (#FF6D00) cho urgency
- **Kích thước:** 24x24px icon, font 13-14px text
- **Format:** SVG để hiển thị sắc nét mọi màn hình

## 4. Lộ Trình Triển Khai 3 Tháng

### Tháng 1 — Nền tảng Trust cơ bản
**Tuần 1-2:**
- [ ] Tạo trang "Về chúng tôi" chuyên nghiệp
- [ ] Viết & publish 3 trang chính sách (bảo hành, đổi trả, vận chuyển)
- [ ] Thêm thông tin doanh nghiệp vào footer (MST, địa chỉ, hotline)
- [ ] Thiết kế & triển khai 5 trust badges

**Tuần 3-4:**
- [ ] Setup hệ thống review sản phẩm (WooCommerce plugin)
- [ ] Thêm Zalo OA chat widget
- [ ] Tạo trang FAQ (câu hỏi thường gặp)
- [ ] Cài đặt SSL certificate (nếu chưa có)

### Tháng 2 — Social Proof & Content Trust
**Tuần 5-6:**
- [ ] Triển khai quy trình thu thập review sau mua hàng
- [ ] Tạo 5 video unboxing sản phẩm bán chạy nhất
- [ ] Chụp ảnh kho hàng, quy trình đóng gói
- [ ] Viết 3 bài blog so sánh hàng chính hãng vs hàng nhái

**Tuần 7-8:**
- [ ] Setup email/Zalo automation xin review
- [ ] Triển khai incentive program (giảm giá cho review)
- [ ] Tạo trang "Khách hàng nói gì?" (testimonials)
- [ ] Thêm counter "X đơn hàng đã bán" trên sản phẩm

### Tháng 3 — Advanced Trust & Conversion
**Tuần 9-10:**
- [ ] Triển khai exit-intent popup với trust message
- [ ] Setup cart abandonment recovery (email + Zalo)
- [ ] A/B test vị trí trust badges
- [ ] Tối ưu checkout flow (giảm bước, thêm trust signals)

**Tuần 11-12:**
- [ ] Đăng ký xác minh trên Bộ Công Thương
- [ ] Tạo press kit, liên hệ báo chí (PR bài viết)
- [ ] Triển khai live chat / chatbot trên website
- [ ] Đánh giá hiệu quả & điều chỉnh chiến lược
