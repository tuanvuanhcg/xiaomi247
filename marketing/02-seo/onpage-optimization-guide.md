### Quy tắc chung
| Yếu tố | Quy tắc |
|---------|---------|
| Title length | 50–60 ký tự (tối đa 70) |
| Meta description | 120–155 ký tự |
| Keyword chính | Đặt ở đầu title tag |
| Brand name | Luôn có "Xiaomi247" ở cuối title |
| CTA trong meta | Luôn có "Xem ngay!", "Đặt mua ngay!", "Đọc ngay!" |
| Unique | Mỗi trang phải có title & meta riêng biệt |

## 2. CẤU TRÚC HEADING H1/H2/H3

### Trang danh mục
```
H1: [Tên danh mục] Xiaomi Chính Hãng — Giá Tốt 2026
  H2: Tại sao chọn [danh mục] Xiaomi?
  H2: Top [danh mục] Xiaomi bán chạy nhất
    H3: [Tên sản phẩm 1]
    H3: [Tên sản phẩm 2]
  H2: Hướng dẫn chọn mua [danh mục] Xiaomi
  H2: Câu hỏi thường gặp về [danh mục] Xiaomi
```

### Trang sản phẩm
```
H1: [Tên sản phẩm đầy đủ]
  H2: Thông số kỹ thuật
  H2: Đánh giá chi tiết [Tên SP]
    H3: Thiết kế & chất lượng xây dựng
    H3: Hiệu năng & tính năng
    H3: Ưu điểm & nhược điểm
  H2: So sánh với sản phẩm tương tự
  H2: Đánh giá từ khách hàng
  H2: Câu hỏi thường gặp
```

### Trang blog
```
H1: [Tiêu đề bài viết — chứa keyword chính]
  H2: [Phần 1 — keyword phụ hoặc LSI]
    H3: Chi tiết phần 1a
    H3: Chi tiết phần 1b
  H2: [Phần 2]
  H2: Kết luận / Tổng kết
  H2: Câu hỏi thường gặp (FAQ)
```

### Quy tắc heading
- Chỉ **1 thẻ H1** trên mỗi trang
- H1 phải chứa **keyword chính**
- H2 nên chứa **keyword phụ hoặc LSI keywords**
- Không bỏ cấp (ví dụ: H1 → H3 mà không có H2)
- Heading phải mô tả chính xác nội dung bên dưới

## 3. INTERNAL LINKING STRATEGY

### Nguyên tắc
- Mỗi trang phải có **ít nhất 3–5 internal links**
- Anchor text phải **chứa keyword** (không dùng "click vào đây")
- Link theo mô hình **hub & spoke** (trang danh mục → sản phẩm, blog → sản phẩm)

### Cấu trúc internal link

| Từ trang | Link đến | Anchor text mẫu |
|----------|----------|-----------------|
| Trang chủ | Danh mục | "Tivi Xiaomi chính hãng", "Robot hút bụi Xiaomi" |
| Danh mục | Sản phẩm | "[Tên SP] — Xem chi tiết", "Mua [Tên SP]" |
| Sản phẩm | Sản phẩm liên quan | "Xem thêm [Tên SP liên quan]" |
| Sản phẩm | Blog review | "Đọc đánh giá chi tiết [Tên SP]" |
| Blog | Sản phẩm | "Mua [Tên SP] chính hãng tại Xiaomi247" |
| Blog | Blog liên quan | "Tham khảo: [Tiêu đề bài liên quan]" |
| Blog | Danh mục | "Xem tất cả [danh mục] Xiaomi" |

### Breadcrumb
- Mọi trang phải có breadcrumb: `Trang chủ > Danh mục > Sản phẩm`
- Implement Schema BreadcrumbList

## 4. IMAGE ALT TEXT GUIDELINES

### Quy tắc
- **Mọi hình ảnh** phải có alt text
- Alt text phải **mô tả hình ảnh + chứa keyword** khi phù hợp
- Độ dài: 5–15 từ
- Tên file ảnh: dùng dấu gạch ngang, có keyword

### Template alt text

| Loại ảnh | Template alt text | Ví dụ |
|----------|-------------------|-------|
| Ảnh sản phẩm chính | "[Tên SP] — [góc chụp/màu sắc]" | "Tivi Xiaomi A Pro 55 inch — mặt trước" |
| Ảnh sản phẩm phụ | "[Tên SP] — [chi tiết]" | "Robot hút bụi Xiaomi X10+ — dock sạc tự đổ rác" |
| Ảnh banner | "[Chương trình/nội dung] tại Xiaomi247" | "Khuyến mãi tivi Xiaomi tháng 3 tại Xiaomi247" |
| Ảnh blog | "[Mô tả nội dung ảnh]" | "So sánh kích thước tivi Xiaomi 43 vs 55 vs 65 inch" |
| Ảnh infographic | "[Chủ đề infographic] — Xiaomi247" | "Infographic cách chọn robot hút bụi phù hợp — Xiaomi247" |

### Tên file ảnh
- ✅ `tivi-xiaomi-a-pro-55-inch-mat-truoc.webp`
- ❌ `IMG_20260301_123456.jpg`
- Format ưu tiên: WebP > JPEG > PNG
- Kích thước: tối đa 200KB cho ảnh sản phẩm, 100KB cho thumbnail

## 6. URL STRUCTURE RECOMMENDATIONS

### Nguyên tắc URL
- Ngắn gọn, dễ đọc, chứa keyword
- Dùng dấu gạch ngang `-` thay dấu gạch dưới `_`
- Chỉ dùng chữ thường, không dấu tiếng Việt
- Không chứa tham số không cần thiết

### Cấu trúc URL theo loại trang

| Loại trang | Format URL | Ví dụ |
|-----------|-----------|-------|
| Trang chủ | `/` | `xiaomi247.com/` |
| Danh mục L1 | `/[danh-muc]/` | `/tivi-xiaomi/` |
| Danh mục L2 | `/[danh-muc]/[phan-loai]/` | `/tivi-xiaomi/55-inch/` |
| Sản phẩm | `/[danh-muc]/[ten-san-pham]/` | `/tivi-xiaomi/a-pro-55-inch-4k/` |
| Blog | `/blog/[slug-bai-viet]/` | `/blog/tivi-xiaomi-co-tot-khong/` |
| Trang tĩnh | `/[ten-trang]/` | `/gioi-thieu/`, `/lien-he/` |
| Khuyến mãi | `/khuyen-mai/[ten-km]/` | `/khuyen-mai/sale-thang-3/` |

### URL cần tránh
- ❌ `/product?id=12345`
- ❌ `/category/subcategory/subcategory2/product-name` (quá sâu)
- ❌ `/tivi-xiaomi-a-pro-55-inch-4k-google-tv-dolby-vision-chinh-hang` (quá dài)
- ✅ `/tivi-xiaomi/a-pro-55-inch-4k/` (ngắn gọn, rõ ràng)
