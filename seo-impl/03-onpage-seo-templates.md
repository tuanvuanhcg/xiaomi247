## 1. TITLE TAG TEMPLATES

### Quy tắc chung

| Quy tắc | Chi tiết |
|---------|----------|
| Độ dài | ≤ 60 ký tự (tối đa 70, nhưng Google cắt ở ~60) |
| Keyword chính | Đặt ở **đầu** title tag |
| Brand name | Luôn có `Xiaomi247` ở **cuối** |
| Separator | Dùng `—` (em dash) hoặc `|` |
| Unique | Mỗi trang phải có title riêng biệt |
| Năm | Thêm năm hiện tại cho trang danh mục & blog |

### Templates theo loại trang

#### 1.1 Trang chủ

```
Format:  Xiaomi247 — [Mô tả ngắn] | [Danh mục chính]
```

| Mẫu | Ký tự |
|------|-------|
| `Xiaomi247 — Đại lý Xiaomi chính hãng | Tivi, Robot, Gia dụng` | 58 |
| `Xiaomi247 — Sản phẩm Xiaomi chính hãng giá tốt nhất` | 52 |

#### 1.2 Trang danh mục

```
Format:  [Keyword danh mục] chính hãng — Giá tốt 2026 | Xiaomi247
```

| Mẫu | Ký tự |
|------|-------|
| `Tivi Xiaomi chính hãng — Giá tốt nhất 2026 | Xiaomi247` | 53 |
| `Robot hút bụi Xiaomi chính hãng — Giá tốt 2026 | Xiaomi247` | 58 |
| `Tủ lạnh Xiaomi chính hãng — Giá tốt 2026 | Xiaomi247` | 53 |

#### 1.3 Trang sản phẩm

```
Format:  [Tên SP] — Giá [giá], Trả góp 0% | Xiaomi247
```

| Mẫu | Ký tự |
|------|-------|
| `Tivi Xiaomi A Pro 55" 4K — Giá 8.990.000đ | Xiaomi247` | 53 |
| `Robot Xiaomi X10+ — Giá 9.490.000đ, Trả góp 0% | Xiaomi247` | 57 |

#### 1.4 Trang blog/bài viết

```
Format:  [Tiêu đề bài viết chứa keyword] — 2026 | Xiaomi247
```

| Mẫu | Ký tự |
|------|-------|
| `Tivi Xiaomi có tốt không? Đánh giá thật — 2026 | Xiaomi247` | 55 |
| `Top 5 robot hút bụi Xiaomi tốt nhất — 2026 | Xiaomi247` | 54 |

#### 1.5 Trang tĩnh (Giới thiệu, Liên hệ, Chính sách)

```

## 3. HEADING STRUCTURE (H1/H2/H3)

### Quy tắc heading

| Quy tắc | Chi tiết |
|---------|----------|
| H1 | Chỉ **1 thẻ H1** trên mỗi trang |
| H1 | Phải chứa **keyword chính** |
| H2 | Chứa **keyword phụ hoặc LSI keywords** |
| Thứ tự | Không bỏ cấp (H1 → H2 → H3, không H1 → H3) |
| Mô tả | Heading phải mô tả chính xác nội dung bên dưới |

### 3.1 Template heading — Trang danh mục

```
H1: [Tên danh mục] Xiaomi Chính Hãng — Giá Tốt 2026
  H2: Tại sao chọn [danh mục] Xiaomi?
  H2: Top [danh mục] Xiaomi bán chạy nhất
    H3: [Tên sản phẩm 1]
    H3: [Tên sản phẩm 2]
    H3: [Tên sản phẩm 3]
  H2: Hướng dẫn chọn mua [danh mục] Xiaomi
    H3: Chọn theo [tiêu chí 1] (kích thước/công suất/...)
    H3: Chọn theo ngân sách
  H2: So sánh [danh mục] Xiaomi với thương hiệu khác
  H2: Câu hỏi thường gặp về [danh mục] Xiaomi
```

**Ví dụ — Trang danh mục Tivi:**
```
H1: Tivi Xiaomi Chính Hãng — Giá Tốt Nhất 2026
  H2: Tại sao chọn Tivi Xiaomi?
  H2: Top Tivi Xiaomi bán chạy nhất 2026
    H3: Tivi Xiaomi A Pro 55 inch 4K
    H3: Tivi Xiaomi A 43 inch
    H3: Tivi Xiaomi A Pro 65 inch
  H2: Hướng dẫn chọn mua Tivi Xiaomi theo kích thước
    H3: Tivi 32-43 inch — Phòng ngủ, phòng bếp
    H3: Tivi 50-55 inch — Phòng khách vừa
    H3: Tivi 65-75 inch — Phòng khách lớn
  H2: So sánh Tivi Xiaomi vs Samsung vs LG
  H2: Câu hỏi thường gặp về Tivi Xiaomi
```

### 3.2 Template heading — Trang sản phẩm

```
H1: [Tên sản phẩm đầy đủ] — Chính hãng Xiaomi
  H2: Thông số kỹ thuật [Tên SP]
  H2: Đánh giá chi tiết [Tên SP]
    H3: Thiết kế & chất lượng xây dựng
    H3: Hiệu năng & tính năng nổi bật
    H3: Ưu điểm & nhược điểm
  H2: Hình ảnh & video [Tên SP]
  H2: So sánh [Tên SP] với sản phẩm tương tự
  H2: Đánh giá từ khách hàng đã mua
  H2: Sản phẩm liên quan
  H2: Câu hỏi thường gặp về [Tên SP]
```

### 3.3 Template heading — Trang blog

```
H1: [Tiêu đề bài viết — chứa keyword chính]
  H2: [Phần 1 — keyword phụ hoặc LSI]
    H3: Chi tiết phần 1a

H1: Tivi Xiaomi có tốt không? Đánh giá thật sau 1 năm sử dụng

## 4. URL STRUCTURE & PERMALINK

### 4.1 Cấu hình WordPress Permalink

**Settings → Permalinks → Custom Structure:**
```
/%category%/%postname%/
```

**Cấu hình WooCommerce:**
- Shop base: `/san-pham/` (ẩn, redirect về danh mục)
- Product permalink: `/[danh-muc]/[ten-san-pham]/`
- Product category base: *(để trống — URL gọn hơn)*

### 4.2 Format URL theo loại trang

| Loại trang | Format URL | Ví dụ |
|-----------|-----------|-------|
| Trang chủ | `/` | `xiaomi247.com/` |
| Danh mục cấp 1 | `/[danh-muc]/` | `/tivi-xiaomi/` |
| Danh mục cấp 2 | `/[danh-muc]/[phan-loai]/` | `/tivi-xiaomi/55-inch/` |
| Sản phẩm | `/[danh-muc]/[ten-sp]/` | `/tivi-xiaomi/a-pro-55-inch-4k/` |
| Blog | `/blog/[slug]/` | `/blog/tivi-xiaomi-co-tot-khong/` |
| Trang tĩnh | `/[ten-trang]/` | `/gioi-thieu/` |
| Khuyến mãi | `/khuyen-mai/[ten-km]/` | `/khuyen-mai/sale-thang-3/` |
| Tag | `/tag/[ten-tag]/` | `/tag/4k/` |

### 4.3 Quy tắc đặt URL

| ✅ Nên | ❌ Tránh |
|--------|---------|
| Chữ thường, không dấu | Chữ hoa, dấu tiếng Việt |
| Dấu gạch ngang `-` | Dấu gạch dưới `_` hoặc dấu cách |
| Ngắn gọn 3-5 từ | Quá dài >8 từ |
| Chứa keyword chính | Chứa tham số `?id=123` |
| Tối đa 2 cấp thư mục | Quá sâu >3 cấp |

### 4.4 URL cần tránh

```
❌ /product?id=12345
❌ /category/subcategory/subcategory2/product-name  (quá sâu)
❌ /tivi-xiaomi-a-pro-55-inch-4k-google-tv-dolby-vision-chinh-hang  (quá dài)
❌ /san-pham/tivi_xiaomi_55  (dấu gạch dưới)
❌ /Tivi-Xiaomi/  (chữ hoa)

✅ /tivi-xiaomi/a-pro-55-inch-4k/
✅ /blog/tivi-xiaomi-co-tot-khong/
✅ /robot-hut-bui-xiaomi/x10-plus/
```

### 4.5 Redirect cần thiết

| URL cũ/sai | Redirect đến | Loại |
|-----------|-------------|------|
| `/san-pham/[tên]` | `/[danh-muc]/[tên]/` | 301 |
| `/danh-muc/[tên]` | `/[tên]/` | 301 |
| URL có dấu tiếng Việt | URL không dấu | 301 |
| URL có chữ hoa | URL chữ thường | 301 |

## 5. IMAGE ALT TEXT & NAMING

### 5.1 Quy tắc chung

| Yếu tố | Quy tắc |
|---------|---------|
| Alt text | **Mọi hình ảnh** phải có alt text |
| Nội dung | Mô tả hình ảnh + chứa keyword khi phù hợp |
| Độ dài | 5–15 từ |
| Tên file | Dùng dấu gạch ngang, có keyword, không dấu |
| Format | WebP > JPEG > PNG |
| Kích thước | Ảnh SP: ≤ 200KB · Thumbnail: ≤ 100KB · Banner: ≤ 300KB |
| Dimensions | Ảnh SP: 800×800px · Banner: 1200×628px · Thumbnail: 400×400px |

### 5.2 Templates alt text theo loại ảnh

| Loại ảnh | Template | Ví dụ |
|----------|---------|-------|
| Ảnh SP chính | `[Tên SP] — [góc chụp/màu sắc]` | `Tivi Xiaomi A Pro 55 inch — mặt trước` |
| Ảnh SP phụ | `[Tên SP] — [chi tiết cụ thể]` | `Robot Xiaomi X10+ — dock sạc tự đổ rác` |
| Ảnh banner | `[Chương trình] tại Xiaomi247` | `Khuyến mãi tivi Xiaomi tháng 3 tại Xiaomi247` |
| Ảnh blog | `[Mô tả nội dung ảnh]` | `So sánh kích thước tivi Xiaomi 43 vs 55 vs 65 inch` |
| Ảnh infographic | `[Chủ đề] — Xiaomi247` | `Infographic cách chọn robot hút bụi — Xiaomi247` |
| Ảnh review | `[Tên SP] — [nội dung review]` | `Tủ lạnh Xiaomi Mijia 430L — bên trong ngăn đông` |

### 5.3 Naming convention file ảnh

**Format:** `[loai-sp]-[ten-sp]-[chi-tiet].[webp]`

| ✅ Đúng | ❌ Sai |
|---------|-------|
| `tivi-xiaomi-a-pro-55-inch-mat-truoc.webp` | `IMG_20260301_123456.jpg` |
| `robot-hut-bui-xiaomi-x10-plus-dock-sac.webp` | `product-1.png` |
| `tu-lanh-xiaomi-mijia-430l-ngan-dong.webp` | `ảnh tủ lạnh.jpg` |
| `may-loc-kk-xiaomi-4-pro-phong-khach.webp` | `DSC_0042.JPG` |
| `banner-khuyen-mai-thang-3-2026.webp` | `banner final (2).png` |

### 5.4 Tối ưu hình ảnh WordPress

```
Plugin khuyến nghị: ShortPixel hoặc Imagify
- Tự động convert sang WebP
- Lazy loading: bật cho tất cả ảnh dưới fold
- Responsive images: WordPress tự tạo srcset
- CDN: Dùng Cloudflare hoặc BunnyCDN cho ảnh
```

## 6. INTERNAL LINKING STRATEGY

### 6.1 Mô hình Hub & Spoke

```
                    ┌─────────────┐
                    │  TRANG CHỦ  │
                    └──────┬──────┘
           ┌───────────────┼───────────────┐
     ┌─────┴─────┐   ┌────┴────┐   ┌──────┴──────┐
     │ Danh mục 1│   │Danh mục 2│  │  Danh mục 3 │  ← HUB pages
     └─────┬─────┘   └────┬────┘   └──────┬──────┘
     ┌─────┼─────┐   ┌────┼────┐   ┌──────┼──────┐
     │SP1  │SP2  │   │SP3 │SP4 │   │SP5   │SP6   │  ← SPOKE pages
     └─────┴─────┘   └────┴────┘   └──────┴──────┘
           ↕               ↕               ↕
     ┌─────────────────────────────────────────────┐
     │              BLOG / CONTENT                 │  ← Liên kết chéo
     └─────────────────────────────────────────────┘
```

### 6.2 Bảng liên kết chi tiết

| Từ trang | Đến trang | Anchor text mẫu | Số lượng link |
|----------|----------|-----------------|---------------|
| Trang chủ | Danh mục Tivi | "Tivi Xiaomi chính hãng" | 1 |
| Trang chủ | Danh mục Robot | "Robot hút bụi Xiaomi" | 1 |
| Trang chủ | Danh mục Tủ lạnh | "Tủ lạnh Xiaomi" | 1 |
| Trang chủ | Danh mục Máy lọc KK | "Máy lọc không khí Xiaomi" | 1 |
| Trang chủ | Danh mục Máy hút ẩm | "Máy hút ẩm Xiaomi" | 1 |
| Trang chủ | Danh mục Gia dụng | "Gia dụng thông minh Xiaomi" | 1 |
| Danh mục | Sản phẩm | "[Tên SP] — Xem chi tiết" | 5–10/trang |
| Danh mục | Blog liên quan | "Hướng dẫn chọn mua [danh mục]" | 2–3/trang |
| Sản phẩm | SP liên quan | "Xem thêm [Tên SP liên quan]" | 3–4/trang |
| Sản phẩm | Blog review | "Đọc đánh giá chi tiết [Tên SP]" | 1–2/trang |
| Sản phẩm | Danh mục cha | "Xem tất cả [danh mục] Xiaomi" | 1/trang |
| Blog | Sản phẩm | "Mua [Tên SP] chính hãng tại Xiaomi247" | 2–3/bài |
| Blog | Blog liên quan | "Tham khảo: [Tiêu đề bài liên quan]" | 2–3/bài |
| Blog | Danh mục | "Xem tất cả [danh mục] Xiaomi" | 1/bài |

### 6.3 Quy tắc internal linking

| Quy tắc | Chi tiết |
|---------|----------|
| Số lượng | Mỗi trang ít nhất **3–5 internal links** |
| Anchor text | Chứa keyword, **không** dùng "click vào đây" |
| Đa dạng | Không dùng cùng 1 anchor text cho nhiều link |
| Contextual | Link nằm trong nội dung, không chỉ sidebar/footer |
| Dofollow | Tất cả internal links đều dofollow |
| Mới → Cũ | Bài mới phải link về bài cũ liên quan |
| Cũ → Mới | Cập nhật bài cũ thêm link đến bài mới |

### 6.4 Breadcrumb

**Cấu trúc breadcrumb cho mọi trang:**

```
Trang chủ > [Danh mục] > [Sản phẩm]
Trang chủ > Blog > [Tiêu đề bài viết]
Trang chủ > [Trang tĩnh]
```

**Plugin:** Yoast SEO hoặc Rank Math (tự động tạo breadcrumb + Schema BreadcrumbList)

## 9. CHECKLIST ON-PAGE SEO

> **Áp dụng cho mỗi trang mới trước khi publish**

### ✅ Title Tag & Meta Description

- [ ] Title tag ≤ 60 ký tự
- [ ] Keyword chính đặt ở đầu title
- [ ] Brand "Xiaomi247" ở cuối title
- [ ] Title unique (không trùng trang khác)
- [ ] Meta description 120–155 ký tự
- [ ] Meta description chứa keyword chính
- [ ] Meta description có CTA (Xem ngay! / Đặt mua ngay!)
- [ ] Meta description unique

### ✅ Heading Structure

- [ ] Chỉ có 1 thẻ H1 trên trang
- [ ] H1 chứa keyword chính
- [ ] H2 chứa keyword phụ / LSI
- [ ] Không bỏ cấp heading (H1→H2→H3)
- [ ] Heading mô tả đúng nội dung bên dưới

### ✅ URL

- [ ] URL ngắn gọn, 3-5 từ
- [ ] URL chứa keyword chính
- [ ] Chữ thường, không dấu tiếng Việt
- [ ] Dùng dấu gạch ngang `-`
- [ ] Tối đa 2 cấp thư mục

### ✅ Nội dung

- [ ] Keyword chính xuất hiện trong 100 từ đầu
- [ ] Keyword density 1-2% (tự nhiên, không nhồi)
- [ ] Nội dung ≥ 300 từ (trang SP), ≥ 1000 từ (blog)
- [ ] Có FAQ section (trang danh mục & blog)
- [ ] Nội dung unique, không duplicate

### ✅ Hình ảnh

- [ ] Mọi ảnh có alt text (5-15 từ)
- [ ] Alt text chứa keyword khi phù hợp
- [ ] Tên file SEO-friendly (dấu gạch ngang, có keyword)
- [ ] Format WebP
- [ ] Kích thước ≤ 200KB (ảnh SP), ≤ 100KB (thumbnail)
- [ ] Lazy loading bật cho ảnh dưới fold

### ✅ Internal Linking

- [ ] Ít nhất 3-5 internal links trên trang
- [ ] Anchor text chứa keyword (không "click vào đây")
- [ ] Link đến sản phẩm liên quan
- [ ] Link đến blog/bài viết liên quan
- [ ] Link đến danh mục cha
- [ ] Breadcrumb hiển thị đúng

### ✅ Schema Markup

- [ ] Schema Product (trang sản phẩm)
- [ ] Schema BreadcrumbList (mọi trang)
- [ ] Schema FAQ (trang danh mục & blog)
- [ ] Schema Organization (trang chủ)
- [ ] Test bằng Google Rich Results Test

### ✅ Technical

- [ ] Canonical URL đúng
- [ ] Open Graph tags (og:title, og:description, og:image)
- [ ] Mobile-friendly (responsive)
- [ ] Page speed ≤ 3 giây
- [ ] Không có broken links
