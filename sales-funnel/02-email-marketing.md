## 1. TẠI SAO EMAIL MARKETING

### 1.1. ROI vượt trội

Email marketing là kênh digital marketing có ROI cao nhất:

| Kênh | ROI trung bình | Chi phí/khách hàng |
| --- | --- | --- |
| Email Marketing | 36:1 (DMA 2024) | ~2.000 VNĐ |
| Facebook Ads | 4-8:1 | 15.000-50.000 VNĐ |
| Google Ads | 2-4:1 | 20.000-80.000 VNĐ |
| TikTok Ads | 2-5:1 | 10.000-40.000 VNĐ |

→ **Cứ 1 đồng chi cho email = 36 đồng doanh thu** (trung bình ngành e-commerce).

### 1.2. Retention vs Acquisition

```
┌──────────────────────────────────────────────────────────┐
│           CHI PHÍ THU HÚT vs GIỮ CHÂN KHÁCH             │
│                                                          │
│  Thu hút khách mới:  ████████████████████ 500K-1M VNĐ   │
│  Giữ chân khách cũ:  ████ 50-100K VNĐ                   │
│                                                          │
│  → Chi phí giữ chân = 1/5 đến 1/10 chi phí thu hút      │
│  → Khách cũ mua lại có conversion rate 60-70%            │
│  → Khách mới chỉ 5-20%                                  │
└──────────────────────────────────────────────────────────┘
```

- **Sản phẩm Xiaomi có lifecycle dài** — Tivi, robot hút bụi dùng 3-5 năm → cần nurture để cross-sell phụ kiện

### 2.2. Khuyến nghị cho Xiaomi247

**Giai đoạn 1: Khởi đầu (0-500 contacts)**

- **Dùng Mailchimp Free** — 500 contacts miễn phí, WooCommerce plugin chính thức, automation cơ bản đủ dùng
- Setup: Welcome series + Cart abandonment + Post-purchase
- Thời gian setup: 1-2 tuần

**Giai đoạn 2: Tăng trưởng (500-2K contacts)**

- **Tiếp tục Mailchimp** — Nâng lên plan Essentials ($13/tháng)
- Thêm: Segmentation, A/B testing, advanced automation
- Đánh giá hiệu quả để quyết định migrate

**Giai đoạn 3: Scale (2K+ contacts)**

- **Migrate sang Klaviyo** — Best-in-class cho e-commerce
- Lý do: Predictive analytics, advanced segmentation, revenue attribution chính xác
- Chi phí: ~$30-100/tháng tùy list size
- ROI expected: Gấp 2-3x so với Mailchimp nhờ personalization tốt hơn

```
TIMELINE MIGRATION:
T1-T2: Mailchimp Free ──→ T3: Mailchimp Essentials ──→ T4+: Klaviyo
(0-500 contacts)           (500-2K contacts)             (2K+ contacts)
```

## 3. LIST BUILDING STRATEGY

### 3.1. Kênh thu thập email

#### a) Homepage Popup (ưu tiên cao nhất)

| Yếu tố | Chi tiết |
| --- | --- |
| Trigger | Delay 10 giây sau khi vào trang |
| Offer | "Giảm 5% đơn hàng đầu tiên — Nhập email để nhận mã!" |
| Design | Popup center, ảnh sản phẩm Xiaomi, form đơn giản (chỉ email) |
| Tần suất | Hiện 1 lần/session, không hiện lại trong 7 ngày nếu đã đóng |
| Mobile | Slide-up từ dưới, chiếm ≤ 30% màn hình (Google compliance) |
| Expected CR | 2-4% visitors → subscribers |
| Plugin | OptinMonster hoặc Popup Maker (WooCommerce compatible) |

**Copy popup mẫu:**

```
🎁 GIẢM NGAY 5% ĐƠN ĐẦU TIÊN

Đăng ký nhận ưu đãi độc quyền từ Xiaomi247
— Sản phẩm mới, Flash Sale, Mã giảm giá —

[Nhập email của bạn]  [NHẬN MÃ GIẢM GIÁ]

✓ Không spam  ✓ Hủy đăng ký bất cứ lúc nào
```

#### b) Exit-Intent Popup

| Yếu tố | Chi tiết |
| --- | --- |
| Trigger | Khi chuột di chuyển ra khỏi viewport (desktop) hoặc scroll up nhanh (mobile) |
| Trang áp dụng | Product pages, Cart page, Category pages |
| Offer (product page) | "Chờ đã! Nhận thông báo khi sản phẩm này GIẢM GIÁ" |
| Offer (cart page) | "Đừng bỏ lỡ! Nhập email để lưu giỏ hàng + nhận mã 5%" |
| Expected CR | 3-5% (cao hơn homepage vì có intent rõ) |

#### c) Checkout Opt-in

| Yếu tố | Chi tiết |
| --- | --- |
| Vị trí | Trước nút "Đặt hàng", sau phần thông tin thanh toán |
| Format | Checkbox: ☑ "Tôi muốn nhận ưu đãi và thông tin sản phẩm mới từ Xiaomi247" |
| Default | Checked by default (opt-out model — phổ biến tại VN) |
| Expected CR | 60-80% (đã có intent mua) |
| Implementation | WooCommerce checkout field customization |

#### d) Blog Content Upgrade

| Nội dung blog | Content upgrade | Format |
| --- | --- | --- |
| "Top 5 Robot Hút Bụi Xiaomi" | Bảng so sánh chi tiết PDF | Inline CTA + popup |
| "Hướng dẫn setup Smart Home" | Checklist setup smart home | Download gate |
| "Review Tivi Xiaomi A Pro 55" | Bảng thông số so sánh các model | Email gate |
| "Cách chọn máy lọc không khí" | Ebook "Hướng dẫn không khí sạch" | Landing page |

**CTA mẫu trong blog:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📥 TẢI MIỄN PHÍ: Bảng so sánh chi tiết 10 Robot Hút Bụi Xiaomi 2026
Nhập email để nhận file PDF ngay:
[Email]  [TẢI NGAY]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

#### e) Zalo OA → Email Collection

- Khi user follow Zalo OA → Auto message: "Nhập email để nhận mã giảm giá 5% + cập nhật khuyến mãi qua email"
- Menu Zalo OA có mục "Đăng ký nhận ưu đãi" → Form thu thập email
- Chatbot Zalo hỏi email khi tư vấn sản phẩm

#### f) Social Media → Landing Page → Email

- Facebook/TikTok posts CTA → Landing page với lead magnet
- Lead magnets: "Top 10 sản phẩm Xiaomi đáng mua nhất 2026", "Catalog Smart Home Xiaomi"
- Facebook Lead Ads → tự động sync với Mailchimp

### 3.2. Target List Growth

| Tháng | Target Subscribers | Kênh chính | Actions |
| --- | --- | --- | --- |
| T1 | 500 | Homepage popup, checkout opt-in | Setup popup + checkout, chạy FB lead ads |
| T2 | 1.000 | + Exit-intent, blog upgrades | Thêm exit-intent popup, publish 4 blog với content upgrade |
| T3 | 2.000 | + Zalo OA, social campaigns | Zalo email collection, FB lead ads scale |

- Xóa subscribers không mở email trong 90 ngày (chuyển sang segment "Dormant" trước)

- Double opt-in cho tất cả sources (trừ checkout)
- Clean list mỗi quý

## 13. A/B TESTING PLAN

### 13.1. Test 1: Subject Line — Emoji vs No Emoji

| Yếu tố | Chi tiết |
| --- | --- |
| Hypothesis | Subject line có emoji sẽ có open rate cao hơn 5%+ |
| Variant A | "Flash Sale — Giảm 30% Robot hút bụi Xiaomi" |
| Variant B | "⚡ Flash Sale — Giảm 30% Robot hút bụi Xiaomi 🔥" |
| Sample size | Min 200 subscribers/variant (total 400) |
| Duration | 4 giờ (đủ để 80%+ opens xảy ra) |
| Winner metric | Open rate |
| Statistical significance | 95% confidence level |
| Timeline | Tuần 2 sau khi launch |

### 13.2. Test 2: Send Time — Sáng vs Tối

| Yếu tố | Chi tiết |
| --- | --- |
| Hypothesis | Email gửi 7PM sẽ có open rate cao hơn vì users VN check email tối |
| Variant A | Gửi lúc 9:00 AM |
| Variant B | Gửi lúc 7:00 PM |
| Sample size | Min 300 subscribers/variant |
| Duration | 24 giờ |
| Winner metric | Open rate + Click rate |
| Timeline | Tuần 3-4 |

### 13.3. Test 3: CTA — Button vs Text Link

| Yếu tố | Chi tiết |
| --- | --- |
| Hypothesis | CTA button sẽ có click rate cao hơn text link |
| Variant A | Text link: "Xem chi tiết tại đây →" |
| Variant B | Button: [XEM CHI TIẾT →] (màu cam, 16px bold) |
| Sample size | Min 300 subscribers/variant |
| Duration | 24 giờ |
| Winner metric | Click-through rate |
| Timeline | Tháng 2 |

### 13.4. Test 4: Layout — 1 Column vs 2 Column

| Yếu tố | Chi tiết |
| --- | --- |
| Hypothesis | 1 column sẽ có click rate cao hơn trên mobile |
| Variant A | 2-column product grid |
| Variant B | 1-column stacked products |
| Sample size | Min 400 subscribers/variant |
| Duration | 48 giờ |
| Winner metric | Click rate + Conversion rate |
| Timeline | Tháng 2 |

### 13.5. Test 5: Personalization — Name vs No Name

| Yếu tố | Chi tiết |
| --- | --- |
| Hypothesis | Dùng tên trong subject line tăng open rate 10%+ |
| Variant A | "Ưu đãi đặc biệt dành cho bạn" |
| Variant B | "[Tên], ưu đãi đặc biệt dành cho bạn" |
| Sample size | Min 200 subscribers/variant |
| Duration | 4 giờ |
| Winner metric | Open rate |
| Timeline | Tháng 2 |

### 13.6. Quy trình A/B Testing

```
1. Đặt hypothesis rõ ràng
   │
2. Chọn 1 biến duy nhất để test
   │
3. Chia audience ngẫu nhiên (50/50)
   │
4. Gửi cùng thời điểm
   │
5. Chờ đủ thời gian (min 4h cho open, 24h cho click)
   │
6. Check statistical significance (95%)
   │
7. Áp dụng winner cho toàn bộ list
   │
8. Document kết quả → test tiếp biến khác
```

**Lưu ý:**

- Chỉ test 1 biến/lần (subject HOẶC CTA HOẶC timing, không kết hợp)
- Sample size tối thiểu 200/variant cho open rate test
- Cần ít nhất 400/variant cho conversion test
- Chạy test ít nhất 2 tuần để có đủ data statistical significance

- ALL CAPS cho toàn bộ subject line
- Quá nhiều !!!!! hoặc $$$$
- "Hành động ngay" quá aggressive

### 15.6. Email Deliverability Tips

1. **Authenticate domain:** Setup SPF, DKIM, DMARC records cho xiaomi247.com
2. **Clean list regularly:** Remove bounced emails ngay, inactive emails mỗi quý
3. **Maintain low bounce rate:** < 2% hard bounce, < 5% soft bounce
4. **Avoid spam traps:** Không mua email list, không import list không xác nhận
5. **Consistent sending:** Gửi email đều đặn, tránh gaps dài rồi gửi nhiều đột ngột
6. **Monitor blacklists:** Check MXToolbox mỗi tuần, resolve ngay nếu bị blacklist
7. **Engagement-based sending:** Gửi nhiều hơn cho subscribers active, ít hơn cho inactive
8. **Sunset policy:** Tự động xóa subscribers không mở email sau 6 tháng
