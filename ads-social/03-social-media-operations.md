## 2. CONTENT CREATION WORKFLOW

### Quy trình 7 bước: Brief → Publish

```
Brief → Draft → Review → Design → Schedule → Publish → Monitor
 D-7     D-5     D-4      D-3      D-1       D-Day    D+1~3
```

| Bước | Timeline | Người thực hiện | Output | Checklist |
|------|----------|----------------|--------|-----------|
| **1. Brief** | D-7 (7 ngày trước) | Social Media Manager | Content brief: chủ đề, mục tiêu, CTA, target audience | ☐ Xác định loại post (sản phẩm/sale/review/engagement) |
| | | | | ☐ Chọn sản phẩm/chủ đề theo calendar |
| | | | | ☐ Xác định KPIs cho post |
| **2. Draft** | D-5 | Content Creator | Caption draft + hashtags | ☐ Viết caption theo template chuẩn |
| | | | | ☐ Chọn hashtags phù hợp (8-10 tags) |
| | | | | ☐ Viết CTA rõ ràng |
| **3. Review** | D-4 | Social Media Manager | Approved caption | ☐ Check chính tả, ngữ pháp |
| | | | | ☐ Verify giá, thông số sản phẩm |
| | | | | ☐ Confirm links hoạt động |
| | | | | ☐ Đảm bảo tone of voice nhất quán |
| **4. Design** | D-3 | Designer | Hình ảnh/Video final | ☐ Ảnh đúng size theo platform |
| | | | | ☐ Text trên ảnh ≤ 20% diện tích |
| | | | | ☐ Logo Xiaomi247 watermark |
| | | | | ☐ Video ≤ 60s (TikTok), ≤ 90s (Reels) |
| **5. Schedule** | D-1 | Social Media Executive | Bài đã schedule | ☐ Schedule đúng giờ tối ưu |
| | | | | ☐ Preview trên từng platform |
| | | | | ☐ Double-check caption + hình ảnh |
| **6. Publish** | D-Day | Social Media Executive | Bài đã live | ☐ Confirm bài đã đăng thành công |
| | | | | ☐ Cross-post các platforms |
| | | | | ☐ Share vào groups liên quan |
| **7. Monitor** | D+1 đến D+3 | Social Media Executive | Performance report | ☐ Track engagement 24h đầu |
| | | | | ☐ Trả lời comments/messages |
| | | | | ☐ Ghi nhận metrics vào báo cáo |

### Content Approval Matrix

| Loại content | Cần approve bởi | Thời gian approve |
|-------------|-----------------|-------------------|
| Post thường (sản phẩm, tips) | Social Media Manager | Trong 24h |
| Flash sale / Khuyến mãi | Manager + Giám đốc | Trong 48h |
| Crisis response | Giám đốc | Ngay lập tức |
| Influencer content | Manager + Giám đốc | Trong 48h |
| UGC repost | Social Media Manager | Trong 12h |
